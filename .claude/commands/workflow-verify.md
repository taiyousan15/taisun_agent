---
description: ワークフロー完了確認 - 全フェーズ完了判定
---

ワークフローが完了したことを確認します。

## 使い方

```bash
npm run workflow:verify
```

## 検証内容

1. すべてのフェーズが完了しているか
2. 最終フェーズの成果物が揃っているか
3. 最終検証がパスしているか

## 重要

「完了しました」と言う前に、必ずこのコマンドを実行してください。
このコマンドがパスしない限り、ワークフローは完了していません。

Verify workflow completion before declaring done.
