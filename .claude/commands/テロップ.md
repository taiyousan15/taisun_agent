# テロップスキル (Telop Skill)

SNS縦型動画（Reels/TikTok/Shorts）向けの高品質テロップ作成スキルです。
Instagram Reels風の多彩なテロップスタイルをRemotionで実装できます。

## 使用方法

```
/テロップ
```

---

## テロップの種類 (11種類)

### 1. フックテキスト (Hook Text)
冒頭で視聴者の注意を引く大型テキスト

```
色: ライム (#CCFF00)
サイズ: 72-96px
効果: 黒アウトライン (3-4px)
用途: 動画冒頭、重要メッセージ
```

### 2. 吹き出し (Speech Bubble)
会話調の導入フレーズ

```
背景: 白
文字: 黒
形状: 角丸 + 尖り
用途: 質問、導入
```

### 3. ボックステキスト (Box Text)
背景付きの説明テキスト

```
背景: パープル (#8B5CF6)
文字: 白 or ライム
角丸: 8-12px
用途: 説明、ポイント
```

### 4. シンプルテキスト (Simple Text)
標準的な字幕

```
色: 白
効果: ドロップシャドウ
用途: 通常の字幕
```

### 5. ブラケットハイライト【】
重要キーワードの強調

```
色: ピンク (#EC4899)
形式: 【テキスト】
用途: 警告、強調
```

### 6. 3D数字 (3D Number)
数値を大きく見せる

```
サイズ: 100-140px
効果: 3D押し出し
用途: 統計、カウント
```

### 7. 3Dテキスト (3D Text)
インパクトのあるCTA

```
色: パープル3D
効果: 押し出し + ひび割れ
用途: CTA、警告
```

### 8. アンダーラインテキスト
セクション見出し

```
装飾: 下線
用途: トピック導入
```

### 9. 段差レイアウト (Staggered)
動的な複数行配置

```
配置: 斜めオフセット
用途: リズム感のある表示
```

### 10. ナンバーバッジ
ステップ番号表示

```
形式: No.1, ①
効果: 3D
配置: 画面端
```

### 11. カラーミックス
一文中の部分強調

```
複数色を1行内で使用
サイズ変更も可能
```

## カラーパレット

| 名前 | HEX | 用途 |
|------|-----|------|
| ライム | #CCFF00 | フック、数字 |
| パープル | #8B5CF6 | ボックス背景 |
| ピンク | #EC4899 | ブラケット |
| レッド | #EF4444 | 警告 |
| ホワイト | #FFFFFF | 標準 |

---

## アニメーション

1. **フェードイン** - 標準の出現
2. **ポップイン** - スケールで弾む
3. **スライドイン** - 横から登場
4. **タイプライター** - 1文字ずつ表示
5. **シェイク** - 震える強調

## Remotion実装例

### フックテキスト
```tsx
<div style={{
  fontFamily: '"Hiragino Sans", "Noto Sans JP", sans-serif',
  fontSize: '80px',
  fontWeight: 900,
  color: '#CCFF00',
  textShadow: '3px 3px 0 #000, -3px -3px 0 #000, 3px -3px 0 #000, -3px 3px 0 #000',
  textAlign: 'center',
}}>
  12月新ルール出ましたね
</div>
```

### ボックステキスト
```tsx
<div style={{
  backgroundColor: '#8B5CF6',
  padding: '16px 24px',
  borderRadius: '10px',
}}>
  <span style={{
    fontFamily: '"Hiragino Sans", sans-serif',
    fontSize: '48px',
    fontWeight: 700,
    color: '#FFFFFF',
  }}>リールの再生回数が急に落ちた</span>
</div>
```

### 3D数字
```tsx
<span style={{
  fontFamily: '"Hiragino Sans", sans-serif',
  fontSize: '120px',
  fontWeight: 900,
  color: '#CCFF00',
  textShadow: '2px 2px 0 #000, 4px 4px 0 #666, 6px 6px 0 #333',
}}>
  10個
</span>
```

## 配置ガイド

```
┌─────────────────────┐
│     上部余白 100px   │
│                     │
│    [メインテロップ]  │ ← 画面下1/3
│                     │
│ [吹き出し] [バッジ]  │ ← 補助要素
│                     │
│     下部余白 150px   │ ← UI避け
└─────────────────────┘
```

---

## ベストプラクティス

1. **1テロップ2行まで**、1行15文字以内
2. **背景とのコントラスト**を確保
3. **強調は1画面1-2箇所**まで
4. **音声と同期**（30fps基準）
5. **動画全体でスタイル統一**
